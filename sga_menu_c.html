Prototipo Menú Versión C

Diseño alternativo con tarjetas claras, nuevo esquema de colores y navegación agrupada por tareas. Uso exclusivo para evaluación de usabilidad en la asignatura de Interacción Humano-Computador.

Bienvenido al prototipo (Versión C)

Utilice el menú de la izquierda para navegar por las diferentes secciones simuladas del SGA. Este diseño prioriza la legibilidad, el agrupamiento por tareas y el uso de tarjetas para ayudar al estudiante a localizar más rápido las opciones.

  • El menú se organiza en tarjetas expansibles por grandes bloques de tarea.
  • Los accesos rápidos se presentan como chips en la parte superior.
  • El área central muestra contenido simulado para cada módulo, útil para el test A/B.
